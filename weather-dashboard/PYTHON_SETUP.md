# Weather Dashboard - Python Backend Setup Guide\n\n## Quick Start (3 Steps)\n\n### Step 1: Install Python Dependencies\n\n```bash\npip install -r requirements.txt\n```\n\n**Alternative (if above doesn't work):**\n```bash\npip install flask requests\n```\n\n### Step 2: Run the Python Script\n\n**Get weather for a city:**\n```bash\npython weather.py Delhi\npython weather.py \"New York\"\npython weather.py Tokyo\n```\n\n**With options:**\n```bash\npython weather.py Delhi --save        # Save to JSON file\npython weather.py Delhi --json        # Show JSON output\npython weather.py --help              # Show help\n```\n\n### Step 3: Run Web Server\n\n```bash\npython server.py\n```\n\nThen open: **http://localhost:5000**\n\n---\n\n## Running in VS Code\n\n### Method 1: Using VS Code Terminal\n\n1. **Open VS Code** and navigate to weather-dashboard folder\n2. **Open Terminal:** `Ctrl + ` (backtick) or View â†’ Terminal\n3. **Install dependencies:**\n   ```bash\n   pip install -r requirements.txt\n   ```\n4. **Run Python script:**\n   ```bash\n   python weather.py Delhi\n   ```\n5. **Or run server:**\n   ```bash\n   python server.py\n   ```\n\n### Method 2: Using Run Configuration\n\n1. Check Python interpreter in VS Code:\n   - Click Python version in bottom right\n   - Select correct Python version\n2.
