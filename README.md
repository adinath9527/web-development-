## Customization

### Change Contact Information
Update the following in `index.html`:
- Phone number in the Contact section
- Email address in the Contact section and form

### Add Your Logo
Replace "StyleHub" text with an image tag:
```html
<div class="logo">
    <img src="your-logo.png" alt="StyleHub">
</div>
```

### Change Colors
Modify the gradient colors in `style.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
